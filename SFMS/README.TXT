%%%%   Socket File Manager System   %%%%



Server: 

Execute the Server.jar inside the Server folder.
Then click Start to start the listening of the program for incoming connections.



Client: 

Execute the Client.jar inside the Client folder. Then you have enter the IP address of your server.
First a user must be registered with password for using the client application. 
Then, use the username and password to login. Now you can create directory, upload files.
And any file or directory can be deleted by it's owner. Every user has his/her personal 
space which is not accessible by others.


Features:

Directory and File Listing
Upload Files
Download Files


Extra Features:

* Create Directory
* Delete Directory
* Large File Upload and Download supported as OS provided restrictions.
